# Typing-speed-test
A Python-based CLI typing speed test that calculates typing speed and errors in real-time using random sentences and time tracking.
Author - Kshitij Srivastava

Characteristics
- A typing test that you can do on the command line
- Calculate typing speed in real time . Finding errors by comparing characters one by one Choose a random sentence for each test. Get instant feedback on your performance.  Keep testing (play multiple times).

Tech Stack: Python 3 Built-in libraries: time for keeping track of how long you type and random for picking random test sentences.

How It Works
The user is asked to start the test (yes or no).
A sentence that is not always the same is shown.
The timer starts as soon as the sentence shows up.
The user types the sentence.
The timer stops when you submit your input.
The program figures out:
Characters typed ÷ Time taken = Typing Speed
Errors happen when the characters in the original and the input don't match.
Results are shown right away.
